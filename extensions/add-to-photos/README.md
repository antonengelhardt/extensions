# Add to Photos

Raycast command that imports **a single image** from the clipboard into the macOS **Photos** library (iCloud Photos when that library is enabled). Multiple images are not supported, as this is possible via the Finder.

**Warning:**: This extension is vibe coded.

## Requirements

- macOS with Photos.app
- **Automation** permission for Raycast to control Photos (macOS will prompt on first import)

## Development

See [Create your first extension](https://developers.raycast.com/basics/create-your-first-extension): `npm install && npm run dev`.
